Learn More

This portfolio is designed to look and function like a desktop enviroment, creating an interactive and engaging way to showcase my work.

Built with pure HTML, CSS and JavaScript.

Users can explore different modals, inclding an 'About Me' window, a calculator and a tic-tac-toe game.
The interface includes dynamic features such as draggable windows, a taskbar displaying the current time, a CPU usage widget and a dropdown menu with customizable settings like theme toggling and brightness adjustment.
